# Travel Fare Calculation System (C++)

This project is a console-based C++ program that calculates travel fares based on transport type, distance, age, baggage weight, travel time, and eligibility for discounts.

## Features
- Supports Bus, Train, and Flight pricing
- Applies age, student, and membership discounts
- Calculates baggage fees and peak-hour surcharges
- Displays a detailed ticket summary

## How to Run
Compile and run using a C++ compiler:

g++ travel_fare.cpp -o travel_fare
./travel_fare
